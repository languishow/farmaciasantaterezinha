# Farmácia Santa Terezinha

Site institucional desenvolvido para a Farmácia Santa Terezinha como trabalho da disciplina de Desenvolvimento de Website.

## 📌 Sobre o projeto

O projeto consiste em uma landing page responsiva para uma farmácia, contendo:

- Página inicial com apresentação da empresa
- Seção “Quem Somos”
- Lista de serviços oferecidos
- Produtos em destaque
- Formulário de contato
- Rodapé com informações da farmácia e dos integrantes do projeto

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Tailwind CSS
- JavaScript
- Google Fonts (Poppins)

## 🎨 Funcionalidades

- Layout responsivo para celular, tablet e computador
- Menu de navegação entre seções
- Cards de produtos com efeito hover
- Formulário de contato interativo
- Alteração do botão após o envio da mensagem
